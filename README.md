# RNK™ Chat Window

**A premium Foundry VTT module by Asgard Innovations**

## Features

Universal system-agnostic module that displays the chat feed in a resizable, draggable window with:

- **Resizable & Draggable Window** - Position and size your chat exactly where you want it
- **12 Beautiful Themes** - Choose from a variety of professionally designed themes
- **Advanced Filtering** - Toggle message types and filter chat content
- **Zoom Controls** - Per-user zoom levels for comfortable reading
- **Auto-Pruning** - Configurable automatic cleanup of old messages
- **Message Retention** - Set maximum message limits
- **Customizable Opacity** - Adjust inactive window opacity
- **Persistent Settings** - Window position and size remembered per client
- **Multilingual Support** - Language support available upon request

## Usage

- Open the RNK™ Chat window from the Modules UI or configured hotkey
- Use header controls to select themes, adjust zoom, and toggle filters
- Configure settings in Foundry's Module Settings panel
- Window position and size are automatically saved

## Compatibility

- **Foundry VTT:** v11 - v13 (verified)
- **System:** Universal (works with all game systems)
